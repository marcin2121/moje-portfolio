# 🚀 Marcin Molenda - Portfolio 2026

---

## 🇺🇸 English Version (EN)

A premium portfolio built with **Next.js**, designed as a highly interactive and fluid user experience (UI/UX) featuring advanced 3D modeling and motion design.

### 🛠️ Tech Stack
- **Framework:** [Next.js](https://nextjs.org/) (App Router, dynamic loading)
- **3D & Canvas:** [@react-three/fiber](https://r3f.docs.pmnd.rs/), [@react-three/drei](https://github.com/pmndrs/drei), [Three.js](https://threejs.org/)
- **Animations:** [GSAP](https://gsap.com/) + [ScrollTrigger](https://gsap.com/docs/v3/Plugins/ScrollTrigger/), [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scroll:** [Lenis](https://lenis.darkroom.engineering/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Utilities:** Lucide Icons, use-sound, Radix UI

### ✨ Key Features
- **3D Interactivity:** The `UrwisModel` integrated into an R3F scene, responsive to interaction and seamlessly blended into the layout.
- **Scroll Snapping & Bento Grid:** Multi-section experience based on smooth horizontal scrolling powered by GSAP.
- **Built-in Demo (Viewport Preview):** A modal for testing other projects (e.g., Sklep Urwis, zamowtu.pl) in **Desktop / Mobile** views directly within the portfolio.
- **Deep Optimization:** Using `dynamic()` and `requestIdleCallback` (rIC) to minimize main thread load.

### 📁 Project Structure
- `/app` – Main page structure (App Router).
- `/components` – Modular components (UI, Hero, UrwisModel).
- `/public` – Static assets (3D models, images, sounds).

### 🚀 Installation
```bash
npm install
npm run dev
```

## 🇵🇱 Wersja Polska (PL)

Portfolio stworzone w technologii **Next.js**, zaprojektowane jako wysoce interaktywne i płynne doświadczenie użytkownika (UI/UX) wykorzystujące zaawansowane modelowanie 3D oraz motion design.

### 🛠️ Stack Technologiczny
- **Framework:** [Next.js](https://nextjs.org/) (App Router, dynamiczne ładowanie)
- **3D & Canvas:** [@react-three/fiber](https://r3f.docs.pmnd.rs/), [@react-three/drei](https://github.com/pmndrs/drei), [Three.js](https://threejs.org/)
- **Animacje:** [GSAP](https://gsap.com/) + [ScrollTrigger](https://gsap.com/docs/v3/Plugins/ScrollTrigger/), [Framer Motion](https://www.framer.com/motion/)
