## 2024-05-18 - [Add ARIA attributes and focus-visible state to FAQ accordion]
**Learning:** React/Framer Motion accordions often lack native accessibility out of the box. Adding explicit `aria-expanded`, `aria-controls` with unique IDs linking toggle buttons to content panels (`role="region"`, `aria-labelledby`) makes them robust for screen readers. Explicit `focus-visible` styles using standard Tailwind classes (`focus-visible:ring-2 focus-visible:ring-orange-500`) drastically improve keyboard navigability without introducing confusing default outlines for mouse users.
**Action:** When inspecting or adding custom interactive components (like accordions or tabs), verify screen reader state linkage (`aria-expanded`, `aria-controls`) and visible focus indicators (`focus-visible`).
